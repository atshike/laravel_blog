# laravel5-blog

基于Laravel5.3开发的简单发布内容博客程序！


## 功能介绍

* 简单文章、分类的增删改查。


## 修改配置文件(.env)

DB_HOST=localhost
DB_DATABASE=laravel_5_3
DB_USERNAME=root
DB_PASSWORD=


##安装

- 下载程序 https://github.com/atshike/laravel_blog.git
- 运行数据库迁移  php artisan migrate:refresh --seed
- 后台地址/login 用户名：atshike 密码：123456
